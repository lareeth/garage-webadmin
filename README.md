# Garage Web Admin

## Administrate your Garage cluster in the browser

**In early development state**

![Preview of the dashboard screen](screenshot-dashboard.png)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Fix code linting and formatting

```sh
npm run fix
```

Or if you want to check for problem without auto fix:

```sh
npm run check
```
