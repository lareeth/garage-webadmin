# User Interface Documentation
A website regrouping interface and visual elements in a documentation space, to help future contributors

https://garage-webadmin-ui-documentation.deuxfleurs.eu

## Project setup

Enter this folder (probably `cd doc-ui`) before entering the commands below

### Install dependencies
You need to have [Node.js](https://nodejs.org/en/download) installed
```
npm install
```
### Launch dev server
```
npm run dev
```
### Build for production
```
npm run build
```
