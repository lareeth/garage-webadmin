A way to split a page into several sub sections and add the ability to navigate between them.
- Every section should have their own url and a change of section should trigger a url change and navigation event
- Keep labels short
- A section should always be selected
- Don't make references of left-right positioning for tab elements since the orders and how they're displayed could change.
