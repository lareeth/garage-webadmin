A way to split a page into several subsections and add the ability to navigate between them.
- Every section should have its own URL and a change of section should trigger a URL change and navigation event
- Keep labels short
- A section should always be selected
- Don't make references of left-right positioning for tab elements, since the orders and how they're displayed could change
