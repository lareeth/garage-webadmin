This website gather guidelines and examples to help maintaining and expanding existing interfaces of the Garage Web Admin project.

## Where to start

The [Design principles](/design-principles) page has global recommendations and mindset advices on how to approach user interfaces on this project.

After that, the [Primitives](/primitives/) category gonna give you informations on the foundational aspects composing almost every elements: [Typography](/primitives/typography/), [Colors](/primitives/colors) and [Icons](/primitives/icons).

## Where to continue

[Navigation](/navigation/) - Help for if i have you have to build entirely new sections or pages or if you have to work on users navigation the app.

[Containers](/containers/) - Blocks made to be composed with elements

[Controls](/controls/) - Actionable elements to controls things in the app

[Feedbacks](/feedbacks/) - Patterns used to inform the users of event or response after an action
