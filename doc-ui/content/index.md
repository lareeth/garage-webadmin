This website gathers guidelines and examples to help maintaining and expanding existing interfaces of the Garage Web Admin project.

## Where to start

The [Design principles](/design-principles) page has global recommendations and mindset advice on how to approach user interfaces on this project.

After that, the [Primitives](/primitives/) category will give you information regarding the foundational aspects composing almost every elements: [Typography](/primitives/typography/), [Colors](/primitives/colors) and [Icons](/primitives/icons).

## Where to continue

[Navigation](/navigation/) - Helpful if you have to build entirely new sections or pages, or if you have to work on user navigation in the app.

[Containers](/containers/) - Blocks made to be composed with elements

[Controls](/controls/) - Actionable elements to controls things in the app

[Feedbacks](/feedbacks/) - Patterns used to inform the users of event or response after an action
